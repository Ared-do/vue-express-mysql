# user-manage-system_demo

## 使用的技术栈
* `element-ui框架`
* `mysql数据库`
* `vue-rouer`
* `token验证`
* `axios发送请求`
## 主要实现功能
* > 用户的增删改查操作
* > 用户的分页管理
* > 根据用户名的查询功能
* > 基于e-charts可视化图表展示
* > element-ui的表格展示
* > tip: 实现的功能比较简单 
## 后台实现(mysql+ express)
* > 常用的sql语句 基于连接池
* > 利用express创建了一个服务器 进行请求的响应操作
